HTML Basics

HTML stands for HyperText Markup Language, the foundation of web structure.

  • <h1> - <h6>: Headings.
  • <p>: Paragraph.
  • <a>: Anchor (link).

Example of a heading:

<h1>This is a heading</h1>
CSS Selectors

CSS Selectors allow you to target HTML elements for styling.

  • .class: Targets all elements with the given class.
  • #id: Targets the element with the given id.
  • element: Targets all specified HTML elements.

Example of a class selector:

p.my-class { color: blue; }
JavaScript Intro

JavaScript adds dynamic behavior to websites.

  • DOM Manipulation.
  • Event handling.
  • AJAX for asynchronous requests.

Declaring a variable:

let x = 10;
DOM Manipulation

The DOM is the structured representation of a document.

With JavaScript, you can modify the DOM dynamically:

document.getElementById("example").innerText = "Updated text";
Responsive Design

Responsive design ensures websites look great on all devices.

Media queries help apply styles based on the device size.

@media (max-width: 600px) { body { background-color: lightblue; } }
